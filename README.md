# Portfolio 2026 - Adrian De Guzman

A modern 3D interactive portfolio website built with Three.js.

## 🚀 Features

- **3D Interactive Model** - Custom GLB model with animations
- **Dual-Layer Navigation** - Innovative stacked navbar with shuffle toggle
- **CRT Shader Effect** - Retro screen overlay with scanlines
- **Responsive Design** - Works on all screen sizes
- **Smooth Animations** 

## 🛠️ Technologies

- **Three.js** - 3D rendering
- **GSAP** - Animations
- **JavaScript (ES6+)** - Core functionality
- **CSS3** - Styling with custom properties
- **HTML5** - Structure
